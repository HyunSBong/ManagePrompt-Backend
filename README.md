# ManagePrompt-Backend
명지대학교 데이터베이스 프로젝트 - 사내 업무용 웹 기반 관리 시스템

## 프로젝트 개요
- 프로젝트 기간: 2023.04.03 ~2023.06.01 (9주)
- 사내 업무용 웹 기반 관리 시스템 개발
- HR, 프로젝트, 평가 관리 시스템 구축  
- 정보의 조회/입력/수정/삭제가 가능한 서비스 제공
- 일반 직원, 관리자(경영진, 인사팀)별 독립적이고 효과적인 업무 수행 기능 제공
